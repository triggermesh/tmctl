## tmctl import

Import TriggerMesh manifest

```
tmctl import -f <path/to/manifest.yaml>/<manifest URL> [flags]
```

### Examples

```
tmctl import -f manifest.yaml
```

### Options

```
  -f, --from string   Import manifest from
  -h, --help          help for import
```
